## 팁: 가상환경(venv) 설정하기

Python 가상환경은 프로젝트마다 필요한 패키지를 따로 관리하기 위한 공간입니다.
과제 폴더에서 아래 순서대로 실행하면 됩니다.

### 1. 가상환경 만들기

```bash
python3 -m venv .venv
```

### 2. 가상환경 활성화하기

macOS/Linux:

```bash
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

활성화가 되면 터미널 앞쪽에 `(.venv)`가 표시됩니다.

### 3. Python 파일 실행하기

```bash
python week02.py
```

### 4. 가상환경 종료하기

```bash
deactivate
```

주의할 점:

- `.venv` 폴더는 GitHub에 올리지 않습니다.
- 필요한 패키지를 설치했다면 다음 명령으로 기록할 수 있습니다.

```bash
pip freeze > requirements.txt
```
